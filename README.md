## Installation

Install Tracker with npm
```bash
  git clone https://github.com/DrunkTaric/Tracker
  cd Tracker
  npm install
```

Install Tracker with pnpm
```bash
  git clone https://github.com/DrunkTaric/Tracker
  cd Tracker
  pnpm install
```
## Deployment

To deploy this on your local machine 

1. Make production build
```bash
    npm run build
```
2. Start the server
```bash
    npm run start
```
